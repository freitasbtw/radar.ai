# Documentação de APIs

Este diretório centraliza a documentação das APIs utilizadas pelo Radar.ai.

## Arquivos

- `EXTERNAL_AUCTION_APIS.md`: documentação técnica das fontes externas de leilão (CAIXA e Receita Federal).
- `SOURCE_REGISTRY_SUPABASE.md`: contrato técnico de source registry com polling, parser e normalização para Supabase.

## Objetivo

Padronizar a forma de integração, normalização e monitoramento das fontes de dados que alimentam o dashboard.
